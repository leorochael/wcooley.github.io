Gollum-site + Jekyll HOWTO

{{ page.title }}

  1. Use local gollum, Github wiki or edit wiki pages.
  2. Use gollum-site to generate pages for Jekyll.
    1. Create _Layout.html with YAML Front Matter for Jekyll, such as the following: 
      ---
title: {{ page.title }}
layout: default
---
{ { page.content }}
  3. Jekyll will then process the pages generated by gollum-site, wrapping them with whatever layout that you have set up.
